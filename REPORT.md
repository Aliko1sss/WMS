# Предметная область Информационная система управления складом
- Репозиторий: https://github.com/Aliko1sss/WMS

## 🎯 Цель модуля
Сформировать практические навыки разработки, интеграции и тестирования программных компонентов.

## 📦 Реализованный функционал (бета-версия)

###  1: Управление товарами (`app/products.py`)
- Создание товара (SKU, название, начальный остаток)
- Получение информации о товаре по ID
- Валидация уникальности SKU

### 2: Складские операции (`app/operations.py`)
- Приход товара (увеличение остатка)
- Расход товара (уменьшение остатка с проверкой наличия)
- История операций (запись в БД)

### Взаимодействие с БД
- СУБД: SQLite (файл `warehouse.db`)
- ORM: SQLAlchemy 2.x
- Модели: `Product`, `Operation` (связь один-ко-многим)
